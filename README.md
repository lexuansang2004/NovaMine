# NovaMine
NovaMine là trợ lý tài chính thông minh theo triết lý Local-first, giúp bảo vệ quyền riêng tư tuyệt đối bằng cách lưu dữ liệu trực tiếp trên thiết bị. Ứng dụng cho phép nhập liệu nhanh qua giọng nói hoặc chụp ảnh hóa đơn để AI tự động xử lý. Với hệ thống báo cáo chi tiết và cơ chế backup an toàn, bạn hoàn toàn làm chủ dòng tiền của mình.
